# Discord Bot Auto-Delete Webhook Messages

This Discord bot automatically deletes messages sent by webhooks after 3 seconds.

## Setup

1. Install dependencies: `npm install`
2. Create a Discord bot and get the token from https://discord.com/developers/applications
3. Replace `YOUR_BOT_TOKEN` in `.env` with your bot token
4. Invite the bot to your server with appropriate permissions (Manage Messages)
5. Run the bot: `npm start`

## Requirements

- Node.js
- Discord bot token
